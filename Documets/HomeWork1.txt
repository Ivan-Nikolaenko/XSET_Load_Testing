Пользовательские сценарии 
- РЖД 
Типичная модели нагрузки за определённый, также типичный отрезок времени для данной системы :
Летний период ,отпуска месяцы (май,июнь,июль,август) дни (сужя по статистике суббота,воскресенье,понедельник и пятница) по времени 10-15МСК.
1) Регистрация на сайте → Поиск интересующего направления с использованием фильтров → Выбор по определенным критериям места в вагоне с использованием UI-интерфейса → Оплата→ Получение подтверждения о покупки и билета на почту пользователя.
2) Авторизация → Переход в ЛК Пользователя → Получение истории покупки биллетов данного пользователя.
3) Авторизация → Переход в ЛК Пользователя → Переход в радел сдачи билета → Ввод требуемой информации по билету → Получение уведомления об супешной → Возврат денежных средств.
- Авито
Типичная модели нагрузки за определённый, также типичный отрезок времени для данной системы :
Зимний период, более преобладают выходные и праздничные дни ,особенно Новогодние каникулы, по времени ближе к вечеру (от 17до22МСК)
1) Поиск товара/услуг через виджет → Получение результата поиска → Выбор товара/услуги → Просмотр карточки продавца и описание товара/услуги
2) Регистрация на площадке → Подтверждение данных пользователя → Размещение товара/услуги 
3) Авторизация → Переход в Лк → Покупка премиума для текущего пользователя
4) Поиск товара/услуг через виджет → Получение результата поиска → Выбор товара/услуги → Просмотр карточки продавца и описание товара/услуги → Нажать Написать сообщение → Произвести диалог с продавцом .
- Авто.ру
Типичная модели нагрузки за определённый, также типичный отрезок времени для данной системы :
Особенная активность на зимний период , максимальный рост продаж это экватор декабрь-январь ,тк у оф дилеров максимальные скидки , по времени Рабочие часы от 10мск до 19
1) Перейти на сайт → Выбрать категорию авто → Применить доступные фильтры (Основные : цена,год,пробег) → Получить результат поиска согласно фильтру 
2) Перейти на сайт → Пройти процедуру регистрации в ЛК → Авторизоваться на сайте
3) Перейти на сайт → Пройти процедуру регистрации в ЛК → Авторизоваться на сайте → поиск авто через виджет → Получение результата поиска → Выбор карточки авто → Просмотр подробного описания → Добавление в избранное.
